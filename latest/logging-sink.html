Sinks#

Define Logging API Sinks.

class gcloud.logging.sink.Sink(name, filter_, destination, client)[source]#

Bases: object

Sinks represent filtered exports for log entries.

See: https://cloud.google.com/logging/docs/api/ref_v2beta1/rest/v2beta1/projects.sinks

Parameters:
  • name (string) – the name of the sink
  • filter (string) – the advanced logs filter expression defining the entries exported by the sink.
  • destination (string) – destination URI for the entries exported by the sink.
  • client (gcloud.logging.client.Client) – A client which holds credentials and project configuration for the sink (which requires a project).
