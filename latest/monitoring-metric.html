class gcloud.monitoring.metric.MetricDescriptor(name, type_, labels, metric_kind, value_type, unit, description, display_name)[source]#

Bases: object

Specification of a metric type and its schema.

Parameters:
  • name (string) – The “resource name” of the metric descriptor. For example: "projects/<project_id>/metricDescriptors/<type>"
  • type (string) – The metric type including a DNS name prefix. For example: "compute.googleapis.com/instance/cpu/utilization"
  • labels (list of LabelDescriptor) – A sequence of label descriptors specifying the labels used to identify a specific instance of this metric.
  • metric_kind (string) – The kind of measurement. It must be one of MetricKind.GAUGE, MetricKind.DELTA, or MetricKind.CUMULATIVE. See MetricKind.
  • value_type (string) – The value type of the metric. It must be one of ValueType.BOOL, ValueType.INT64, ValueType.DOUBLE, ValueType.STRING, ValueType.DISTRIBUTION, or ValueType.MONEY. See ValueType.
  • unit (string) – The unit in which the metric value is reported.
  • description (string) – A detailed description of the metric.
  • display_name (string) – A concise name for the metric.
